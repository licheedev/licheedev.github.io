Android Studio 2.0 的即时运行（Instant Run）太赞了

就不久前，Android Studio才出了1.5正式版，我还没用上AS1.5，昨天又看到说出了AS 2.0预览版了。

http://www.jianshu.com/p/eb860d8709b5

据说性能有很大提升，有新的模拟器，还有屌炸天的即时运行（Instant Run）。

于是晚上就把AS给升级了，简单的耍了下。

不过暂时没看到所谓的新模拟器，可能是还没上线，过几天再看。

而性能方面，说实话，首次编译的时候还是那么卡（Windows平台），卡到鼠标指针偶尔还是会停顿几下。挣方面没怎么觉得有啥大变化。

不过下面这个 Instant Run 真心逆天了，上图：

就是这么爽，再也不用改一点点代码，就又得等几分钟才能编译好了。

顺便期待一下那个新模拟器，毕竟Genymotion免费版的功能有点弱，原官方的模拟器又太卡。
